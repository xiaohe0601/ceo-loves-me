# The CEO Loves Me: A Front-End Developer Story

霸道女总裁爱上前端开发的我

[🎮 在线体验](https://ceo-loves-me.netlify.app)

## 项目概述

一款基于 Web 的剧情互动游戏，讲述一名 24 岁的前端开发工程师在加入新公司后，通过一系列职场与情感事件，
最终获得年轻漂亮女总裁青睐的故事。

## 游戏特性

- 完整的叙事结构，包含 6 个章节和 4 种不同结局
- 逐行对话系统，提供沉浸式剧情体验
- 数值系统（好感度、自信度），影响游戏结局
- 多分支选择系统，玩家决策决定剧情走向
- 精美的 UI 设计，浪漫主义配色
- 响应式布局，支持桌面和移动设备
- 流畅的动画效果和交互体验

## 游戏玩法

1. 在开始界面输入你的称呼
2. 阅读剧情对话，点击「继续」按钮推进剧情
3. 在关键节点做出选择，影响好感度和自信度
4. 根据最终数值触发不同结局：
   - **情定终身**: 好感度 ≥ 90 且 自信度 ≥ 80
   - **携手同行**: 好感度 ≥ 70 且 自信度 ≥ 60
   - **初识之缘**: 好感度 ≥ 50 且 自信度 ≥ 40
   - **擦肩而过**: 其他情况

## 技术栈

- 开发语言: TypeScript
- 构建工具: Vite
- 代码规范: ESLint + TypeScript
- 包管理器: PNPM
